In this paragraph we are trying to explain with few words what the below steps are about. It is about the HTML5 and CSS3. Let's follow the steps below and start a web page from scratch.

First steps to HTML 5

Choose a text editor

There are a lot of text editors to choose. You can use notepad++ for Windows OS or TextEdit for Mac users. What text editor you use will not effect your work at all.

Save the file with .html extension

Open up the text editor of your choice and save the file with .html extension and begin to start writing your web page.

Start with DOCTYPE

When starting to write html, always start with <!DOCTYPE html> then follow with html tags, head tags then the body tags

Tags and lowercase

In HTML5, most of the tags must be closed (for example, <html> tag becomes </html>). Write all the tags in lowercase letters but the DOCTYPE.

Save and check

It is a good practice to save your work and check periodically so to avoid mistakes and make sure that everything looks accordingly. Also it is easier to correct small mistakes early on.

Keep notes

It is a very good practice to keep notes inside the HTML files. In order to comment we can use this sign " <!-- place your notes inside this signs--> "

First Steps with CSS 3

The steps in the unordered list below explain how to write External CSS3 codes and connect it with the HTML page.

  • Open a text editor

  • Save the file with .css extension

  • The code is placed inside "{}"

  • CSS works in Property - Value pair

  • Link it with the HTML file

  • Place both HTML & CSS files in one folder

  • Keep notes

    • It is a very good practice to keep notes inside the CSS files too. In order to comment we can use this signs /* place your notes inside these signs */
